Anleitungen

iOS (iPhone/iPad)

  1. Kopieren Sie den obigen Link.
  2. Öffnen Sie die "Einstellungen" App.
  3. Gehen Sie zu "Kalender" -> "Accounts" -> "Account hinzufügen".
  4. Wählen Sie "Andere" -> "Kalenderabo hinzufügen".
  5. Fügen Sie den Link ein und folgen Sie den Anweisungen.

macOS

  1. Öffnen Sie die "Kalender" App.
  2. Wählen Sie "Ablage" -> "Neues Kalenderabonnement".
  3. Fügen Sie den Link ein und klicken Sie auf "Abonnieren".

Android

Die einfachste Methode ist, den Kalender über Ihren Google-Account hinzuzufügen:

  1. Öffnen Sie den Google Kalender im Webbrowser auf einem Computer.
  2. Klicken Sie neben "Weitere Kalender" auf das "+"-Symbol und wählen Sie "Per URL".
  3. Fügen Sie den Link zum iCS-Kalender ein.
  4. Der Kalender wird automatisch mit der Google Kalender App auf Ihrem Android-Gerät synchronisiert.

Windows

  1. Öffnen Sie die "Kalender" App.
  2. Klicken Sie auf "Kalender hinzufügen" -> "Aus dem Web".
  3. Fügen Sie den Link zum iCS-Kalender ein.
